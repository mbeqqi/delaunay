+ les fichiers sources sont sous ./code/

+ afin de compiler le projet, lancer build.bat en utilisant un environement où les chemins des outils de compilations VC++2015 sont bien définis.
+ le programme génère à chaque lancement un ensemble aléatoire de points.
+ Appuyez sur 'Echap' pour appliquer la triangulation naive
+ Appuyez sur 'ENTRER' pour appliquer la triangulation de delaunay.
+ vous pouvez faire bouger les points avec la souris (bouton cliqué + mouvement)

- La structure de données utilisée est le HalfEdge (définie sous 'mesh.h').
